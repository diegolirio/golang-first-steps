# Go Tour 

## Install 

You must download the specific file for your OS.
https://go.dev/dl/
   
And then you can create a `file.go` with the content as bellow:
```go 
package main

import "fmt"

func main() {
	fmt.Println("Hello, 世界")
}
```

Execute with command-line:
```sh 
go run your-file.go
```

Other commands:
```sh 
go build file.go

```
